Getting Started

Back to Sample Overview.

Run the renderer

deno run -A ./render-doc-html.ts \
  --input ./sample/docs \
  --output ./sample/output/index.html \
  --title "Sample Docs"

After rendering, open ./sample/output/index.html in a browser.
