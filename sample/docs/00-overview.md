# Sample Overview

This sample shows a tiny Markdown/JSON docs tree that can be rendered into a
single static HTML documentation site.

## Included examples

- a relative Markdown link to [Getting Started](./guide/getting-started.md#run-the-renderer)
- a local JSON document link to [Profile JSON](./data/profile.json)
- a regular code block with syntax highlighting
- a Mermaid diagram

## Example code block

```ts
const sampleTitle = "Sample Docs";
console.log(`Rendering ${sampleTitle}`);
```
